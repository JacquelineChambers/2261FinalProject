SPLASH SCREEN
- START - goes to instructions
- SELECT - goes straight to game

INSTRUCTIONS
- SELECT - goes to splash screen
- START -goes to game

GAME
- SELECT goes to lose/win state
- B - goes to cutscene (do not press)
- START - goes to pause

PAUSE
- START - goes back to game
- SELECT - goes to splash screen

WIN/LOSE
-START - goes back to splash screen
